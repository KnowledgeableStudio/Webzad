# Webzad Official

This folder contains the final publish-ready website only.

Files included:
- `index.html` live homepage entry
- `404.html` static-hosting fallback
- `assets/` built JavaScript, CSS, and images
- `favicon.png` site icon
- `.nojekyll` GitHub Pages compatibility file

This package does not require the source project files.
Upload the contents of this folder as the root of your GitHub repository if you want to publish the live website directly.
